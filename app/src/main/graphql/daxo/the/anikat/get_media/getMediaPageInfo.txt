Параметры для настройки отображения

type:
ANIME: для поиска аниме.
MANGA: для поиска манги.

format:
TV: Телевизионное аниме.
TV_SHORT: Короткие эпизоды.

MOVIE: Фильмы.
OVA: OVA (Original Video Animation).
ONA: ONA (Original Net Animation).

SPECIAL: Спецвыпуски.
MUSIC: Музыкальные клипы.

status:
FINISHED: Завершённые.
RELEASING: В процессе выхода.
NOT_YET_RELEASED: Еще не вышедшие.
CANCELLED: Отменённые.
HIATUS: Временно приостановленные.

season:
WINTER: Зима.
SPRING: Весна.
SUMMER: Лето.
FALL: Осень.

seasonYear:
Например, 2023 (для фильтрации по году).

genre:
Например, Action, Romance, Comedy, и так далее (фильтрация по жанру).

search:
Строка поиска для фильтрации по названию.

isAdult:
true или false (для фильтрации по взрослому контенту).

tag:
Например, Isekai, Shounen, Slice of Life и т.д. (фильтрация по тэгам).

Виды сортировки (sort)
POPULARITY_DESC / POPULARITY: Сортировка по убыванию или возрастанию популярности.
SCORE_DESC / SCORE: Сортировка по убыванию или возрастанию среднего рейтинга (оценки).
TRENDING_DESC / TRENDING: Сортировка по убыванию или возрастанию трендовости (насколько аниме/манга активно обсуждается).
TITLE_ROMAJI_DESC / TITLE_ROMAJI: Сортировка по убыванию или возрастанию названия в транскрипции ромадзи.
TITLE_ENGLISH_DESC / TITLE_ENGLISH: Сортировка по убыванию или возрастанию названия на английском.
TITLE_NATIVE_DESC / TITLE_NATIVE: Сортировка по убыванию или возрастанию названия на оригинальном языке (например, на японском).
EPISODES_DESC / EPISODES: Сортировка по убыванию или возрастанию количества эпизодов.
START_DATE_DESC / START_DATE: Сортировка по убыванию или возрастанию даты начала трансляции.
END_DATE_DESC / END_DATE: Сортировка по убыванию или возрастанию даты окончания трансляции.
FAVOURITES_DESC / FAVOURITES: Сортировка по убыванию или возрастанию количества добавлений в избранное.
ID_DESC / ID: Сортировка по убыванию или возрастанию уникального идентификатора.
